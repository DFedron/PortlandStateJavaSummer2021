This Java program is about making an appointment. It can create an appointment,
and add it to appointment book. There is three buttons on the main page, create new appointment,
search by owner name, and search by time. You can hint the create new appointment button to enter
the page that can create appointment. Search by owner name button can bring you to a page that can
search all of information in appointment book by owner name. Last button can search all of appointments
information between the beginTime and endTime that you provide.

Below is some information that you may provide.
owner               The person who owns the appt book
description         A description of the appointment
beginTime           When the appt begins
endTime             When the appt ends

Date and time should be in the format: mm/dd/yyyy hh:mm am/pm
